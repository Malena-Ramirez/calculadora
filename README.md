## Instalación

1. Clona el repositorio
   ```sh
   git clone https://github.com/Malena-Ramirez/calculadora.git
   ```
3. Intala los paquetes de NPM
   ```sh
   npm install
   ```

## 📌 Stack

- React
- Styled components
- Mathjs
